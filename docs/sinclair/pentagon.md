# Pentagon 128K

- **`make MACHINE=pentagon`** — Sinclair
- **Year**: 1991
- **Manufacturer**: Vladimir Drozdov
- **Television**: PAL

## At power-on

A Russian Spectrum clone, boots to a 128-style startup menu (Tape Loader, 128 BASIC, Calculator, 48 BASIC, TR-DOS) on the PAL canvas.

## Required assets

- `roms/pentagon.zip`

  | ROM | CRC32 |
  |---|---|
  | `128p-0.rom` | `124ad9e0` |
  | `128p-1.rom` | `b96a36be` |
- `roms/spec128.zip` — the shared 128 parent romset
- `roms/betadisk.zip` — Beta Disk / TR-DOS interface ROMs

## Notes

- Its built-in Beta Disk interface carries a TR-DOS entry on the startup menu.
