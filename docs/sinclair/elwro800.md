# Elwro 800-3 Junior

- **`make MACHINE=elwro800`** — Sinclair
- **Year**: 1986
- **Manufacturer**: Elwro
- **Television**: PAL

## At power-on

A Polish Spectrum clone built for schools, boots to an `ELWRO 800-3 Junior` banner (`64kB RAM 24kB ROM   wersja dyskowa`, the yellow `elwro` logo, the Spectrum K cursor) on the PAL canvas.
