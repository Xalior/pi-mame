# MicroART ATM-Turbo 2

- **`make MACHINE=atmtb2`** — Sinclair
- **Year**: 1992
- **Manufacturer**: MicroART
- **Television**: PAL

## At power-on

A Russian turbo Spectrum clone, boots to a MicroART firmware menu (CP/M, TR-DOS 48, Spectrum 128, Spectrum 48, Turbo On) over a red MicroART logo on the PAL canvas.

## Required assets

- `roms/atmtb2.zip`

  | ROM | CRC32 |
  |---|---|
  | `atm106.rom` | `75350b37` |
  | `rf2ve3.rom` | `35e0f9ec` |
  | `rfat710.rom` | `03734365` |
  | `sgen.rom` | `1f4387d6` |
- `roms/spec128.zip` — the shared 128 parent romset
- `roms/betadisk.zip` — Beta Disk / TR-DOS interface ROMs
