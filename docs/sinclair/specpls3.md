# ZX Spectrum +3

- **`make MACHINE=specpls3`** — Sinclair
- **Year**: 1987
- **Manufacturer**: Amstrad plc
- **Television**: PAL

## At power-on

ZX Spectrum +3 startup menu (Loader, +3 BASIC, Calculator, 48 BASIC; drives A: and M:) — the same firmware with the built-in 3" floppy drive.
