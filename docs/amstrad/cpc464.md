# Amstrad CPC464

- **`make MACHINE=cpc464`** — Amstrad
- **Year**: 1984
- **Manufacturer**: Amstrad plc
- **Television**: PAL

## At power-on

Amstrad's all-in-one home computer, boots to Locomotive BASIC 1.0: the yellow-on-blue `Amstrad 64K Microcomputer (v1)` / `©1984 Amstrad Consumer Electronics plc and Locomotive Software Ltd.` sign-on over `BASIC 1.0` / `Ready`, on the PAL canvas.
