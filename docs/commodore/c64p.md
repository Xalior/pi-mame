# Commodore 64 (PAL)

![Commodore 64 (PAL) at power-on](images/c64p.jpg)

- **`make kernel MACHINE=c64p`** — Commodore Business Machines
- **Year**: 1982
- **Manufacturer**: Commodore Business Machines
- **Television**: PAL

## At power-on

Commodore 64 BASIC V2, `READY.` — the IEC disk bus boots empty (`-iec8
""`), so no drive romset is required to reach BASIC. Same BASIC as the
NTSC c64; only the video/CIA timing is PAL.

## Required assets

- `roms/c64p.zip`

  | ROM | CRC32 |
  |---|---|
  | `901226-01.u3` (basic) | `f833d117` |
  | `901227-03.u4` (kernal r3) | `dbe3e7c7` |
  | `901225-01.u5` (chargen) | `ec4272ee` |
  | `906114-01.u17` (PLA) | `54c89351` |

  Byte-identical to the NTSC `c64` romset (`rom_c64p` is defined as
  `rom_c64` in the driver) — the difference is machine timing, not ROM data.
